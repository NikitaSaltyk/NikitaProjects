Инструкция по синхронизации.

1. Создаем Репозитарий с таким же именем в ГитХаб без файла РИДМИ.
2. Вводим последовательно стедующие операции, где после имени пользователя идет имя репозитария:
1)git remote add origin https://github.com/NikitaSaltyk/NikitaProjects.git 
2)git branch -M main 
3)git push -u origin main
3. Обновляем репозитарий в ГитХаб



Инструкция по обновлению данных после редактирования файла в Гит

1.Редактируем файл, сохраняем
2. Вводим git status
    - появилась красная строка с измененным файлом
3. Вводим git add (тут либо ставим точку git add . чтобы обновились все файлы, либо имя конкретного файла
4. Дальше git push (может не сработать)
4. Дальше git commit 
   - нужно будет ввести комментарий на английском, какие изменения сделаны в последней версии, нажать esc и два раза shift+Z
   Появится строка сколько файлов изменено и тп.
5. Дальше возможно git push, если не сработает, то первая инструкция
